# The Syntax Atelier · 句法工坊

> 翻译不是搬运砖头（单词），而是搬运蓝图（逻辑）。

一款帮助任何想学好英语的人，通过"逻辑解构"真正读懂英文长难句的 AI 交互工具。

---

## 核心功能

- **解构模式**：输入任意英文长句，AI 实时拆解骨架、逻辑色块、三阶译文
- **挑战模式**：AI 根据你学过的句子动态出题，测试解构思维
- **学习档案**：追踪学习盲点，可视化进步，解锁成就

## 技术栈

- React + Tailwind CSS
- Claude API（解构引擎）
- IndexedDB / Dexie.js（本地数据存储）
- Cloudflare Pages / Vercel（部署）

## 开发模式

Vibe Coding — 意图优先，最短路径，数据归用户，完全开源。
