## 4. 关键技术信息

| 项目 | 内容 |
|------|------|
| 默认 AI 引擎 | 智谱 GLM-4-Flash |
| API 端点（GLM）| https://open.bigmodel.cn/api/paas/v4/chat/completions |
| 支持模型 | GLM-4-Flash/Air/Plus、Gemini 2.0 Flash、GPT-4o mini、Claude Haiku、Grok-3 mini |
| env 变量 | VITE_GLM_API_KEY（默认GLM Key） |
| 用户自定义 Key | localStorage，字段名：apiKey_glm / apiKey_{modelId} |
| 本地存储 | IndexedDB via Dexie.js，version 4 |
| 表结构 | analyses、quizRecords、achievements |
| 线上地址 | https://the-syntax-atelier.pages.dev |
| GitHub | https://github.com/stonesun2023/the-syntax-atelier |
| 部署方式 | Cloudflare Pages，GitHub push 后自动部署 |
| 开发端口 | localhost:5173（或5175） |
| 开发浏览器 | Chrome |

## 5. IndexedDB 表结构（version 4）

**analyses**：解构历史记录
**quizRecords**：答题会话记录（含 sentence、date、results数组、score、accuracy）
**achievements**：已解锁成就（含 achievementId、unlockedAt）

成就ID列表：`first_analysis`、`century_sentences`、`imagery_master`、`modifier_hunter`、`inversion_terminator`
