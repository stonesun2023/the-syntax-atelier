## 5. 第二阶段任务说明

### 核心目标
用户在解构模式里学过的句子，AI 自动出题，形成学练闭环。

### 出题 API
generateQuiz(sentence, analysisResult) 已在 claudeApi.js 写好，返回4道题的 JSON 数组。

### 需要新建的文件
- src/components/ChallengeMode.jsx     # 挑战模式主容器
- src/components/QuizCard.jsx          # 单题交互组件
- src/components/QuizResult.jsx        # 完成页面

### 交互设计参考
参考 Gemini 案例（见产品设计方案 v2.0 第三节）：
- 顶部进度条：当前题号 / 总题数
- 实时对错计数（红色×，绿色✓）
- 选择后立即显示正确答案 + 原理解释
- 完成页：得分 / 准确率 / AI 表现分析
